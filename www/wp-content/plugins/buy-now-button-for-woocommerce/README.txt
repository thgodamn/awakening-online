Customers expect a fast and seamless shopping experience. Give shoppers the easiest way to make a purchase. The Buy Now Button for WooCommerce will help achieve this goal.

== Description ==

After installing the plugin, next to the "add to cart" button, an additional button will appear: "Buy now". Increase sales by skipping a cart with the Buy Now button. Or provide the standard Add to Cart process. Send users directly to the checkout after clicking on Buy now button! Fewer steps to checkout - boost your business conversion rate: remove a time-consuming cart page loading. And all this in a plugin: Buy Now Button for WooCommerce.

== Installation ==

1. Unzip the downloaded zip file and upload the plugin folder into the `wp-content/plugins/` directory. Alternatively, upload from Plugins >> Add New >> Upload Plugin.
2. Activate the plugin through the 'Plugins' menu in WordPress
